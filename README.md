# Super Mars工作室

一个顾客只需输入对应房间号即可加入的简易聊天网站。

支持发送文字和图片。可以点击“图片”按钮选择图片、把图片直接拖入聊天窗口，或在输入框使用 `Ctrl+V` 粘贴截图。图片会在浏览器中自动压缩后进入预览区，点击发送后发出；聊天文字采用较小字号，适合电脑和手机使用。

## 固定网站部署

项目已经包含云端部署版本：

- `server.js`：零第三方依赖的 Node.js 服务端
- `package.json`：云平台启动配置
- `render.yaml`：Render Blueprint 配置
- `Dockerfile`：通用容器部署配置

在 Render 创建 Web Service 后，使用 `node server.js` 启动，即可获得固定的 HTTPS 网站地址。健康检查地址是 `/health`。

如果使用 Render 免费实例，长时间无人访问时服务可能会休眠，聊天记录也可能在重新部署后丢失。正式营业建议选择长期运行实例并配置持久磁盘，将环境变量 `DATA_DIR` 设置为持久磁盘目录。

## 启动

双击 `启动聊天室.bat`。它会启动服务器，并自动在浏览器打开：

```text
http://localhost:8080
```

请保留标题为 `Super Mars Studio Server - DO NOT CLOSE` 的服务器窗口；关闭它后聊天室也会停止。

同一 Wi-Fi / 局域网中的朋友，可以打开启动窗口显示的“同一网络打开”地址。

第一次让其他设备访问前，请右键或双击运行 `首次使用-允许局域网访问.bat`，并允许管理员权限。其他设备必须连接同一个路由器，使用启动窗口显示的局域网地址，不能使用 `localhost`。

## 通过互联网聊天

双击 `互联网启动聊天室.bat`，首次启动时会从 Cloudflare 官方发布地址下载隧道工具。等待窗口中出现以 `trycloudflare.com` 结尾的 HTTPS 地址，把该地址发给朋友即可。地址也会保存在 `public-address.txt`。

公网地址在每次启动时会变化，关闭互联网启动窗口后地址立即失效。使用此模式无需配置路由器，也无需开放 Windows 防火墙入站端口。

知道公网地址和房间号的人可以进入聊天，请只把地址和房间号发给信任的人。Cloudflare Quick Tunnel 适合测试和临时使用，不适合长期正式部署。

## 数据

聊天记录保存在 `data/messages.json`，最多保留最近 3000 条消息。
